# FitTrain Telegram Bot

Лёгкий, но полезный Telegram-бот для напоминаний о тренировках, отметки выполнения и простой аналитики.

## Что умеет MVP v2

- многопользовательский режим
- стандартные и пользовательские типы тренировок
- напоминания:
  - по дням недели
  - разовое
  - через интервал в днях
- действия по напоминанию:
  - выполнить
  - пропустить
  - перенести
- follow-up через 30 минут
- вечерняя проверка
- ручная запись тренировки
- аналитика за неделю и месяц
- кнопка `Что сегодня`

## Быстрый запуск

### 1. Установи Python 3.11+

### 2. Создай бота у BotFather
Получи токен и вставь его в `.env`.

### 3. Подготовь окружение

```bash
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

Linux/macOS:

```bash
source .venv/bin/activate
```

### 4. Установи зависимости

```bash
pip install -r requirements.txt
```

### 5. Создай `.env`
Скопируй `.env.example` в `.env` и заполни токен:

```env
BOT_TOKEN=123456:ABCDEF...
DATABASE_URL=sqlite+aiosqlite:///fittrain.db
BOT_TIMEZONE=Europe/Moscow
ADMIN_IDS=123456789
EVENING_CHECK_HOUR=20
FOLLOWUP_DELAY_MINUTES=30
```

### 6. Запуск

```bash
python run.py
```

## Что важно знать

- бот работает по московскому времени `Europe/Moscow`
- база по умолчанию: `SQLite`
- при первом запуске таблицы создаются автоматически
- стандартные типы тренировок тоже создаются автоматически
